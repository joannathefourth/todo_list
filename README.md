# Todo List App 

A Simple react todo list application to help manage your tasks. 

## Features 

 - Add new todos
 - Mark todos as completed 
 - Delete todos


## Getting started 
1. clone the repo: 
    git clone https://github.com/your-username/todo_list.git
2. install dependencies:
    npm install 
3. Start the app: 
    npm start
